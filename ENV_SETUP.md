# Environment Variables Template

Copy this file to `.env.local` and fill in your values:

```
NEXT_PUBLIC_SANITY_PROJECT_ID=your-project-id
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_API_TOKEN=your-api-token
```

To get your Sanity credentials:
1. Create a new Sanity project: `npm create sanity@latest`
2. Or visit https://sanity.io/manage to find your project ID
3. The dataset is usually `production` by default
4. API token can be created in Sanity Dashboard → Settings → API → Tokens
